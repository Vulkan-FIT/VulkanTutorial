Other useful resources

  • Vulkan specification - I recommend the core only specification for inital study, or the one with KHR extensions (with ratified extensions) as it contains WSI (Window System Integration). WSI is implemented using extensions so it is missing in core only specification. All the remaining versions of Vulkan specification are available at Khronos Vulkan Registry including html versions.
  • Khronos Vulkan Tutorial - using C Vulkan API and GLFW
  • Website of vulkan.org, especially Key resources section and Vulkan Tutorials.
  • Vulkan tutorial on root.cz - Czech language only, using vulkan.hpp, source code on github.
